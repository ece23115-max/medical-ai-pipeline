# Multi-Paradigm Medical AI Diagnostic Pipeline

## 📌 Project Overview
An end-to-end Machine Learning and Natural Language Processing pipeline built using Object-Oriented Programming principles. The architecture cascades simulated patient records through multiple algorithmic paradigms to accurately predict metabolic phenotypes, classify unstructured clinical intake notes, and map latent disease progression over time.

## 🛠️ Tech Stack & Tools
* **Programming Language:** Python
* **Machine Learning & Data Science:** Scikit-Learn, Pandas, NumPy
* **Algorithmic Models:** PCA, Gaussian Mixture Models (GMM), Multi-Layer Perceptron (MLP), Multinomial Naive Bayes, Hidden Markov Models (HMM), Conditional Random Fields (CRF), MEMM

## 🚀 Key Features
* **Phenotype Profiling:** Applies PCA for feature reduction and GMM clustering to locate hidden patient risk groups.
* **Clinical Notes Triage:** Implements a Generative NLP module using Count Vectorizer and Naive Bayes to triage unstructured text logs.
* **Sequence Modeling:** Tracks multi-stage disease paths using dynamically trained Markov and CRF sequential probability structures.
